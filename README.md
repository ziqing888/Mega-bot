## 安装和使用

1. 下载代码库：
    ```bash
    git clone https://github.com/ziqing888/Mega-bot.git
    cd Mega-bot
    ```
2. 安装依赖：
    ```bash
    npm install
    ```
3. 运行脚本：
    ```bash
    npm run start
    ```
4. 根据终端中的提示完成设置。
